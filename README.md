# LangChain Python Project

This project demonstrates a basic usage of the LangChain library in Python.

## Setup

1. Ensure you have Python 3.7 or higher installed.
2. A virtual environment is already set up in this folder.
3. Required package `langchain` is already installed.

## Usage

Run the example script:

```
.venv\Scripts\python.exe langchain_example.py
```

## Example

The included script `langchain_example.py` shows a simple LangChain chain in action.
